# NetworkWeaver Design System: "NetSec Pro"

## 1. Core Philosophy
- **Minimalist**: Reduce visual noise. Content first.
- **Professional**: High contrast, precise alignment, subtle depth.
- **Tech-Forward**: Dark mode default, teal/slate palette.

## 2. Color Palette

### Backgrounds
- **Surface 0 (Main)**: `#0f172a` (Slate 950)
- **Surface 1 (Card)**: `#1e293b` (Slate 800)
- **Surface 2 (Hover)**: `#334155` (Slate 700)
- **Overlay**: `rgba(0,0,0,0.5)`

### Accents (Functional)
- **Primary (Action)**: `#0ea5e9` (Sky 500)
- **Secondary (Focus)**: `#64748b` (Slate 500)
- **Success**: `#10b981` (Emerald 500)
- **Warning**: `#f59e0b` (Amber 500)
- **Danger**: `#ef4444` (Red 500)

### Typography
- **Headings**: `#f8fafc` (Slate 50)
- **Body**: `#cbd5e1` (Slate 300)
- **Muted**: `#94a3b8` (Slate 400)

## 3. Shadows & Effects
- **Card Shadow**: `0 4px 6px -1px rgba(0, 0, 0, 0.3)`
- **Glow (Subtle)**: `0 0 15px rgba(14, 165, 233, 0.1)`
- **Border**: `1px solid rgba(148, 163, 184, 0.1)`

## 4. Components

### Cards
- Background: Surface 1
- Border: 1px solid Surface 2 (transparent)
- Radius: `0.75rem` (12px)

### Buttons
- **Primary**: Solid Primary, White Text, No Border.
- **Ghost**: Transparent, Primary Text, Hover Surface 2.

### Inputs
- Background: Surface 0
- Border: 1px solid Surface 2
- Focus: Border Primary
